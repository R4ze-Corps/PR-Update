# ECOSYSTEM UPDATE — Bot de PROMOÇÃO

**Visão Geral**

Bot para gerenciamento de hierarquia de membros com promoções e rebaixamentos, atribuição automática de cargos complementares e notificações em canal designado.

**Comando**

**`/upar`**

Abre painel com **Promover** e **Rebaixar**.

**Promoção:** selecione o cargo (Elite, Gerentes, 01-03) → selecione o membro. Se for gerência, o cargo complementar (`Gerência`) é adicionado automáticamente.

**Rebaixamento:** selecione o cargo (Gerentes, Membro) → selecione o membro. Se **não** for gerência, o cargo @Gerência é removido.

**Cargos de @Gerência**

Os cargos abaixo ativam a regra do cargo @Gerência:

- Gerente de Ação, Gerente de Elite, Gerente de Vendas
- Gerente de Farm, Gerente de Recrutamento, Gerente Geral

**Como Usar**

No canal <@1502487831170383974> Utilize o botão **Promover** ou **Rebaixar**

- Para promover selecione o cargo para a promoção, após isso selecione o membro que será promovido.
- Para rebaixar selecione o novo cargo que ele será rebaixado, após isso selecione o membro que será rebaixado.
